Changing Themes

Using jQuery

We will create a jQuery code so if the user clicks on Light Theme button, you will apply the appropriate class to make the whole page lighter. The image below will also be changed to show the image of the sun! If the Dark Theme button is clicked, the image of the moon will be displayed and the appropriate class will be applied to the whole page as well

            <select name="city" id="city" hx-trigger="load, changed" hx-post="/api/search" hx-target="#result">
                <option>San Francisco</option>
                <option>New York</option>
                <option>Las Vegas</option>
            </select>
  • "/api/search" within hx-post attribute represents the URL to send the post request
  • To allow for multiple triggers types in htmx we would need to add hx-trigger="load, changed"
  • To output results I would create a div with id result and add hx-target="#result"
